# Canteen Ordering System

A Kivy-based application that allows users to browse menus and place orders.

## Features
Vendor:
- Menu editing
- Order checking
- Updating orders

Student:
- Menu browsing
- Order submission

## Tech Stack
- Python
- Kivy

## Setup(Important)
This project requires a Google Cloud service account file.

Steps:
1. Go to Google Cloud Console
2. Create a service account
3. Download the JSON key
4. Rename it to:
   service_account.json
5. Place it in the project root folder

## How to Run
- pip install -r requirements.txt
- python student_app.py/vendor_app.py
